

# 表情包生成系统 V1.0 接口文档 

## 1. 全局规范

### 1.1 通信协议

- **请求方式**：统一使用 `POST`
- **Content-Type**：`application/json` （文件上传除外）
- **鉴权方式**：请求头携带 `Authorization: Bearer <token>`

### 1.2 统一响应格式

```json
{
  "code": "LC000",          // 业务状态码，200为成功，非200为异常
  "message": "success", // 提示信息
  "data": {}            // 业务数据
}
```

### 1.3 统一分页参数

**请求参数：**

| 参数名   | 类型    | 必填 | 说明              |
| -------- | ------- | ---- | ----------------- |
| pageNum  | Integer | 否   | 当前页码，默认 1  |
| pageSize | Integer | 否   | 每页条数，默认 20 |

**响应 `data` 结构：**



```json
{
  "list": [],       // 数据列表
  "total": 100,     // 总记录数
  "pageNum": 1,     
  "pageSize": 20    
}
```

## 2. 角色管理模块 (对应 `cloud_emoji_ip_info`)

### 2.1 AI 扩写角色提示词

- **URL**: `/emoji/ip/expand-prompt`

- 请求参数

  ：

  | 参数名  | 类型   | 必填 | 说明                  |
  | ------- | ------ | ---- | --------------------- |
  | keyword | String | 是   | 角色关键词（≤50字符） |

- 响应数据 (`data`)

  | 参数名     | 类型   | 说明                   |
  | ---------- | ------ | ---------------------- |
  | promptText | String | 扩写后的英文图像提示词 |

### 2.2 保存角色 IP

- **URL**: `/emoji/ip/save`

- 请求参数

  | 参数名        | 类型    | 必填 | 说明                                 |
  | ------------- | ------- | ---- | ------------------------------------ |
  | name          | String  | 是   | 角色名称                             |
  | imageUrl      | String  | 是   | 角色图片 URL                         |
  | sourceType    | String  | 是   | 来源：`ai_generated` / `user_upload` |
  | sourceKeyword | String  | 否   | 用户输入的角色关键词（AI生成时必填） |
  | promptText    | String  | 否   | AI扩写后的提示词（AI生成时必填）     |
  | description   | String  | 否   | 角色描述（≤1000字符）                |
  | isDefault     | Integer | 否   | 是否默认：0-否，1-是，默认0          |

- 响应数据 (`data`)

  | 参数名 | 类型   | 说明       |
  | ------ | ------ | ---------- |
  | ipId   | String | 角色 IP ID |

### 2.3 获取我的角色列表

- **URL**: `/emoji/ip/list`

- 请求参数

  | 参数名     | 类型    | 必填 | 说明               |
  | ---------- | ------- | ---- | ------------------ |
  | name       | String  | 否   | 按角色名称模糊搜索 |
  | sourceType | String  | 否   | 按来源类型筛选     |
  | pageNum    | Integer | 否   | 页码               |
  | pageSize   | Integer | 否   | 每页条数           |

- 响应数据 (`data.list` 元素)

  ：严格映射 

  ```
  cloud_emoji_ip_info
  ```

   字段

  | 参数名        | 类型    | 说明         |
  | ------------- | ------- | ------------ |
  | ipId          | String  | 角色 ID      |
  | name          | String  | 角色名称     |
  | imageUrl      | String  | 角色图片 URL |
  | sourceType    | String  | 来源类型     |
  | sourceKeyword | String  | 角色关键词   |
  | promptText    | String  | 图像提示词   |
  | description   | String  | 角色描述     |
  | isDefault     | Integer | 是否默认角色 |
  | createTime    | String  | 创建时间     |

### 2.4 更新角色信息

- **URL**: `/emoji/ip/update`

- 请求参数

  | 参数名      | 类型    | 必填 | 说明                     |
  | ----------- | ------- | ---- | ------------------------ |
  | ipId        | String  | 是   | 角色 ID                  |
  | name        | String  | 否   | 新的角色名称             |
  | description | String  | 否   | 新的角色描述             |
  | isDefault   | Integer | 否   | 是否设为默认：0-否，1-是 |

### 2.5 删除角色

- **URL**: `/emoji/ip/delete`

- 请求参数

  | 参数名 | 类型   | 必填 | 说明    |
  | ------ | ------ | ---- | ------- |
  | ipId   | String | 是   | 角色 ID |

## 3. 表情包系列管理 (对应 `cloud_emoji_group`)

### 3.1 创建表情包系列

- **URL**: `/emoji/group/create`

- 请求参数

  | 参数名         | 类型   | 必填 | 说明                             |
  | -------------- | ------ | ---- | -------------------------------- |
  | ipId           | String | 是   | 关联的角色 IP ID                 |
  | seriesKeywords | String | 是   | 系列关键词（逗号分隔，≤100字符） |
  | emojiType      | String | 是   | 生成类型：`static` / `dynamic`   |
  | emojiStyle     | String | 否   | 表情风格（如“可爱”、“写实”）     |

- 响应数据 (`data`)

  | 参数名       | 类型   | 说明                |
  | ------------ | ------ | ------------------- |
  | emojiGroupId | String | 创建成功后的系列 ID |

### 3.2 获取我的系列列表

- **URL**: `/emoji/group/list`

- 请求参数

  | 参数名         | 类型    | 必填 | 说明             |
  | -------------- | ------- | ---- | ---------------- |
  | seriesKeywords | String  | 否   | 按系列关键词搜索 |
  | emojiType      | String  | 否   | 按类型筛选       |
  | ipId           | String  | 否   | 按角色筛选       |
  | pageNum        | Integer | 否   | 页码             |
  | pageSize       | Integer | 否   | 每页条数         |

- 响应数据 (`data.list` 元素)

  ：严格映射 

  ```
  cloud_emoji_group
  ```

   字段，并联表统计数量

  | 参数名         | 类型    | 说明                                  |
  | -------------- | ------- | ------------------------------------- |
  | emojiGroupId   | String  | 系列 ID                               |
  | ipId           | String  | 关联角色 ID                           |
  | seriesKeywords | String  | 系列关键词                            |
  | emojiType      | String  | 类型                                  |
  | emojiStyle     | String  | 风格                                  |
  | emojiCount     | Integer | 该系列下已生成的表情包数量 (联表统计) |
  | createTime     | String  | 创建时间                              |

### 3.3 获取系列详情

- **URL**: `/emoji/group/detail`
- **请求参数**：`emojiGroupId` (String, 必填)
- **响应数据 (`data`)**：返回 `cloud_emoji_group` 全部字段。

### 3.4 删除表情包系列

- **URL**: `/emoji/group/delete`
- **请求参数**：`emojiGroupId` (String, 必填)
- **说明**：逻辑删除系列，同时级联逻辑删除该系列下的所有 `cloud_emoji_info` 资产。

## 4. 表情包生成与资产管理 (对应 `cloud_emoji_info`)

### 4.1 批量生成并保存表情包

- **URL**: `/emoji/emoji/batch-generate`

- **说明**：触发异步生成任务，生成成功后自动写入 `cloud_emoji_info` 表。

- 请求参数

  | 参数名       | 类型   | 必填 | 说明                                          |
  | ------------ | ------ | ---- | --------------------------------------------- |
  | emojiGroupId | String | 是   | 系列 ID                                       |
  | ipId         | String | 是   | 角色 IP ID                                    |
  | descriptions | Array  | 是   | 选中的描述列表对象数组                        |
  | ↳ name       | String | 是   | 表情名称                                      |
  | ↳ promptText | String | 是   | **前端拼接好的完整提示词** (含情境/动作/元素) |

- 响应数据 (`data`)

  | 参数名 | 类型   | 说明                          |
  | ------ | ------ | ----------------------------- |
  | taskId | String | 批量生成任务 ID，用于轮询进度 |

### 4.2 重新生成单个表情包

- **URL**: `/emoji/emoji/regenerate`

- 请求参数

  | 参数名  | 类型   | 必填 | 说明      |
  | ------- | ------ | ---- | --------- |
  | emojiId | String | 是   | 表情包 ID |

### 4.3 查询生成任务进度

- **URL**: `/emoji/emoji/task-progress`
- **请求参数**：`taskId` (String, 必填)
- **响应数据 (`data`)**：包含 `total`, `successCount`, `failedCount`, `status`。

### 4.4 获取表情包资产列表

- **URL**: `/emoji/emoji/list`

- 请求参数

  | 参数名       | 类型    | 必填 | 说明                                        |
  | ------------ | ------- | ---- | ------------------------------------------- |
  | name         | String  | 否   | 按表情名称搜索                              |
  | type         | String  | 否   | 按类型筛选：static / dynamic                |
  | emojiGroupId | String  | 否   | 按所属系列筛选                              |
  | promptText   | String  | 否   | 按提示词内容模糊搜索 (替代原需求的情境搜索) |
  | pageNum      | Integer | 否   | 页码                                        |
  | pageSize     | Integer | 否   | 每页条数                                    |

- 响应数据 (`data.list` 元素)

  ：严格映射 

  ```
  cloud_emoji_info
  ```

   字段

  | 参数名           | 类型    | 说明                                                  |
  | ---------------- | ------- | ----------------------------------------------------- |
  | emojiId          | String  | 表情包 ID                                             |
  | name             | String  | 表情名称                                              |
  | promptText       | String  | 完整图像提示词                                        |
  | type             | String  | 类型                                                  |
  | fileUrl          | String  | 文件 URL                                              |
  | fileSize         | Integer | 文件大小 (KB)                                         |
  | emojiGroupId     | String  | 所属系列 ID                                           |
  | createTime       | String  | 创建时间                                              |
  | latestSyncStatus | String  | **最新同步状态** (联表 `cloud_emoji_sync_tasks` 获取) |

### 4.5 删除表情包资产

- **URL**: `/emoji/emoji/delete`
- **请求参数**：`emojiIds` (Array, 必填，支持批量)

## 5. 平台同步模块 (对应 `cloud_emoji_sync_tasks`)

### 5.1 创建平台同步任务

- **URL**: `/emoji/sync/create-task`

- 请求参数

  | 参数名         | 类型   | 必填 | 说明                                                         |
  | -------------- | ------ | ---- | ------------------------------------------------------------ |
  | targetPlatform | String | 是   | 目标平台：`wechat` / `douyin`                                |
  | emojiGroupId   | String | 否   | 关联系列 ID（按系列同步时传）                                |
  | syncIds        | Array  | 否   | 表情包 ID 数组（对应表结构中的 `sync_id`，按单张/多选同步时传） |
  | metaData       | Object | 否   | 同步元数据（名称、简介、标签等）                             |

- 响应数据 (`data`)

  | 参数名 | 类型   | 说明        |
  | ------ | ------ | ----------- |
  | taskId | String | 同步任务 ID |

### 5.2 确认微信手动上传完成

- **URL**: `/emoji/sync/confirm-wechat`

- **说明**：微信 V1.0 半自动方案，用户手动上传后点击确认，更新任务状态为 success。

- 请求参数

  | 参数名 | 类型   | 必填 | 说明            |
  | ------ | ------ | ---- | --------------- |
  | taskId | String | 是   | 微信同步任务 ID |

### 5.3 获取同步任务状态

- **URL**: `/emoji/sync/task-status`

- **请求参数**：`taskId` (String, 必填)

- 响应数据 (`data`)

  ：严格映射 

  ```
  cloud_emoji_sync_tasks
  ```

   字段

  | 参数名           | 类型   | 说明                                          |
  | ---------------- | ------ | --------------------------------------------- |
  | taskId           | String | 任务 ID                                       |
  | targetPlatform   | String | 目标平台                                      |
  | taskStatus       | String | 状态：pending / processing / success / failed |
  | platformResponse | String | 平台返回信息或失败原因                        |
  | createTime       | String | 任务创建时间                                  |

### 5.4 获取我的同步任务列表

- **URL**: `/emoji/sync/task-list`
- **请求参数**：支持 `targetPlatform`, `taskStatus` 筛选及分页参数。
- **响应数据**：返回 `cloud_emoji_sync_tasks` 列表。

## 6. 通用基础模块

### 6.1 文件上传

- **URL**: `/emoji/common/upload`

- **请求格式**：`multipart/form-data` (唯一不使用 JSON 的接口)

- 请求参数

  | 参数名  | 类型   | 必填 | 说明                                 |
  | ------- | ------ | ---- | ------------------------------------ |
  | file    | File   | 是   | 文件对象 (jpg/png/webp/gif, ≤10MB)   |
  | bizType | String | 是   | 业务类型：`ip_image` / `emoji_asset` |

- 响应数据 (`data`)

  | 参数名   | 类型    | 说明          |
  | -------- | ------- | ------------- |
  | fileUrl  | String  | 文件访问 URL  |
  | fileSize | Integer | 文件大小 (KB) |
